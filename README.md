## ✨ What's Inside

A fully custom, single-file portfolio with:

- **WebGL "Dark Veil" shader** — animated gradient background running on the GPU via a canvas element
- **Typing animation** — role text cycles through on the hero section
- **Scrolling skills marquee** — infinite CSS animation, no JavaScript library
- **Stats bar** — internships, hackathon wins, projects, year at a glance
- **Projects section** — waste scanner app and resort management system with stack details
- **Experience section** — Future Interns (Full Stack) and Inamigos Foundation (Data Analytics)
- **Contact form** — wired to Formspree, works without a backend
- **Fully responsive** — mobile hamburger menu, fluid layouts

---

## 🛠️ Tech Used

| Layer | Detail |
|---|---|
| Structure | HTML5 (semantic) |
| Styling | CSS3 — custom properties, Flexbox, Grid, keyframe animations |
| Interactivity | Vanilla JavaScript — WebGL shader, scroll effects, nav toggle |
| Fonts | DM Serif Display · Inter · JetBrains Mono (Google Fonts) |
| Form | Formspree (no backend needed) |
| Hosting | GitHub Pages |

Zero npm. Zero build step. Zero dependencies installed. Open the file and it runs.

## 🚀 Running Locally

```bash
git clone https://github.com/YuvanRanjith/portfolio.git
cd portfolio
# Just open the file — no server needed
open index-final.html
```

Or drag the file into any browser.
